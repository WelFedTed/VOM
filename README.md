# VOM-BARF (Vigilant Output Minimizer)

Batch wrapper for BARF compressor

## Usage

**Compress File**
> vom c input</br>
> </br>
> Test Example:</br>
> vom c test.txt

**Decompress File**
> vom d input</br>
> </br>
> Test Example:</br>
> vom d test.txt.x2u.x3x.x4e.x4e.x4h.x1g.x3j.x4h.x4k.x4e.x3w.x1h

**Re-Create Included Test file (text.txt)**
> test
